What is a variable?

A variable is a named container used to store data so it can be used and reused throughout a program.

What is a data type?

A data type describes what kind of value a variable holds — text, numbers, true/false, and more.

Variables

Store data with names you can remember

Variable Basics

Every variable has a name and a value. Variables save data so you can reference it later.

let studentName = "Asha";
let marks = 95;

Declaration Keywords

var let const

var — older, function-scoped.

let — block-scoped, can be reassigned.

const — block-scoped, cannot be reassigned.

Naming Rules

  • Use letters, numbers, _, and $.
  • Cannot start with a number.
  • Cannot use reserved keywords.
  • Prefer meaningful names like totalPrice.

Scope

Scope controls where a variable is accessible.

let and const are block-scoped.

var is function-scoped.

Reassignment

You can update a variable's value when it's declared with let or var.

let city = "Delhi";
city = "Mumbai";       // ✓ allowed

const country = "India";
// country = "Japan";  // ✗ not allowed

Data Types

Understand what kind of value is stored

Primitive Data Types

Basic types that hold a single, immutable value.

StringNumberBoolean UndefinedNullBigIntSymbol

Non-Primitive Data Types

Complex types used to group and structure multiple values.

ObjectArrayFunctionDate

Primitive Examples

let age      = 21;        // Number
let name     = "Riya";   // String
let isPassed = true;     // Boolean
let data;                // Undefined
let empty    = null;     // Null

Non-Primitive Examples

let colors = ["red", "blue"];

let student = {
  name: "Riya",
  age: 21
};

function greet() {
  return "Hello!";
}

Type Lab

Check and convert values

Type Checking

Use typeof to inspect a value's type at runtime.

typeof "Hello"  // "string"
typeof 45       // "number"
typeof true     // "boolean"

Type Conversion

Convert between types explicitly using built-in functions.

Number("25")   // 25
String(100)    // "100"
Boolean(1)     // true
